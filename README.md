# AdEditor

Standalone Vue 3 low-code H5 page editor MVP.

## Stack

- Vue 3
- TypeScript
- Vite
- Pinia
- Tailwind CSS
- Element Plus

## Scripts

```bash
npm install
npm run dev
npm run test
npm run typecheck
npm run build
```

Open `/editor` for the editor and `/preview` for the H5 preview page.
